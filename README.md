- 📊 Track wellness trends

- ⚡ Fast, responsive, and user-friendly  

---

## 🛠️ Tech Stack  
- **Framework:** Flutter  
- **Language:** Dart  
- **IDE:** Android Studio 

---

## 📥 Installation  

1. Clone the repository:  
   git clone https://github.com/yashgrover25/health_checkin_app.git

2. Navigate to the project directory:
   cd health_checkin_app
   
3. Get dependencies:

   flutter pub get

  
4. Run the app:

   flutter run
